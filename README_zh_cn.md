# BeatmapDownloader

一个游戏内谱面下载器

## 如何使用

1. 从[此处](https://github.com/KyuubiRan/BeatmapDownloader/releases)下载最新版本
2. 解压zip文件
3. 运行 `osu!.exe`
4. 运行 `Injector.exe`
5. 完工！
