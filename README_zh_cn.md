# BeatmapDownloader

一个游戏内谱面下载器  
[介绍视频](https://www.bilibili.com/video/BV1kP411i7An) by JunMoyan

## 如何使用

1. 从[此处](https://github.com/KyuubiRan/BeatmapDownloader/releases)下载最新版本
2. 解压zip文件
3. 运行 `Injector.exe`
4. 选择 `osu!.exe`
5. 完工！
