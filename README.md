# BeatmapDownloader
An in-game beatmap downloader for osu!stable

## Usage
1. Download the latest release from [here](https://github.com/KyuubiRan/BeatmapDownloader/releases)
2. Extract the zip file
3. Run `osu!.exe`
4. Run `Injector.exe`
5. Enjoy!
